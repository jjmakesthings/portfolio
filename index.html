Spring-Damper Physics Simulator

In mechanical engineering, a spring-damper model is often used to analyze shocks and isolators. These models are described by diferential equations that are best solved with numberical methods. This desktop application allows a user to slect a type of system and input a number of required and optional variables to solve for motion of a spring-damper system.

The program accepts test data in .xls format, and allows for saving or loading a serialized .pkl file that describes the state of every input. After a simulation is run, the data is displayed graphically in a seperate window and may also be exported to a .xls file.

Python Tkinter Matplotlib

Script to Categorize and Design 432 Bridge Damper Designs

A a script used to design and sort bridge cable dampers for a project that required 432 dampers with unique specifications. The script loads customer data into a Pandas dataframe, performs minimum sizing calculations for each location, groups the designs, and assigns complete part numbers. This enables the designer to experiment with grouping thresholds to create the smallest number of unique designs while minimizing overengineering.
